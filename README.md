**Requirements:**

1.Python3.7+
2.Django
3.DRF
4.djangorestframework-simplejwt
5.django-taggit
6.django-taggit-serializer

**Instructions**

Download this repository.

1.Navigate to Project directory(cd Ajackus_Task/Ajackus_Assignment/)
2.Make sure you have Python3.7+ installed.
3.Install pip3
4.command: pip3 install -r requirements.txt
5.command: python manage.py makemigrations
6.command: python manage.py migrate
7.command: python manage.py seed
8.command: python manage.py runserver

**Note: Admin users and City,State,Country are created using seeding**
